# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmph6wQu2/file243c36e5b43e.shp"
templaz()
#> [1] "/tmp/Rtmph6wQu2/file243c2290e7ce.laz"
```
