# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmp2NZf4a/file243c460c39d.shp"
templaz()
#> [1] "/tmp/Rtmp2NZf4a/file243c7ac440a8.laz"
```
