# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/Rtmp8rZ8W0/file24d0214a93d1.shp"
templaz()
#> [1] "/tmp/Rtmp8rZ8W0/file24d08e35654.laz"
```
