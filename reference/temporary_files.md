# Temporary files

Convenient functions to create temporary file with a given extension.

## Usage

``` r
temptif()

tempgpkg()

tempshp()

templas()

templaz()

temppcd()
```

## Value

string. Path to a temporary file.

## Examples

``` r
tempshp()
#> [1] "/tmp/RtmpKOZPyh/file25a530e3605e.shp"
templaz()
#> [1] "/tmp/RtmpKOZPyh/file25a54c95763a.laz"
```
